# Generated TypeScript SDK types

Auto-generated from `mashgate/contracts/proto/v1/*.proto` через:
- `protoc + protoc-gen-openapi` → `openapi.yaml`
- `openapi-typescript` → `types.ts`

**Не редактируй файлы вручную.** Регенерация: `make sdk-gen-ts` в mashgate.

Pipeline: `mashgate/scripts/sdk-gen-ts.sh`.
